Weighted Views Transformer

Project Summary

This project developed a Transformer-based pipeline in TensorFlow to predict molecular properties, achieving second-to-state-of-the-art performance on the QM7 dataset. We engineered a novel geometry-based featurization method to create robust molecular views, achieving a test MAE of 9.46. The pipeline leverages weighted views, atomic embeddings, and a customized Transformer Encoder to enhance feature association and prediction accuracy.

To optimize performance, We conducted extensive hyperparameter tuning, exploring configurations of encoder blocks, attention dimensions, and feedforward layers. The model was trained on the Ohio Supercomputer Center’s Pitzer cluster, ensuring scalability for complex molecular data. Further results and algorithms will be published upon thesis release.

Challenges and Insights

Developing this molecular property prediction pipeline presented several challenges:

  • Variable-Sized Molecular Inputs: Molecules in the QM7 dataset vary in size, complicating Transformer input processing. We implemented zero-padding in the Transformer Encoder (qm7_transformer_padded_zero.ipynb), ensuring consistent input shapes while preserving feature integrity, significantly improving model robustness.
  • Balancing View Weights: Initial models struggled with assigning appropriate weights to molecular views, leading to suboptimal predictions. By developing a geometry-based featurization method in qm7_weightedviews.py, We enhanced view relevance, boosting the model’s ability to capture molecular structures effectively.
  • Hyperparameter Sensitivity: The Transformer’s performance was highly sensitive to parameters like d_k, d_v, and d_ff. Manual tuning was inefficient, so We systematically explored configurations (as shown in the README table), achieving a test MAE of 9.46 with d_k=32, d_v=16, d_q=32, d_ff=64, and 3 blocks.
