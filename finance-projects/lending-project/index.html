Lending Club Loan Prediction

1) Summary

This project uses a neural network to predict loan status (Fully Paid vs. Charged Off) using real Lending Club data. It includes preprocessing, feature engineering, model design using TensorFlow/Keras, and evaluation using AUC and precision-recall.

The goal is to provide a reproducible, end-to-end machine learning pipeline that models credit risk using real-world borrower data. All code is available on GitHub.

2) Project Motivation

Peer-to-peer lending has revolutionized the finance industry by allowing borrowers and investors to interact without traditional banks. However, the risk of loan default remains a significant concern. Financial institutions and investors alike need tools to assess borrower risk before issuing loans.

Lending Club provides a rich dataset that mirrors real-world lending behavior, making it an ideal candidate for a machine learning project. My goal in this project was to apply neural networks to this dataset to accurately predict loan outcomes. This not only demonstrates how AI can support financial decision-making but also provides valuable insight into borrower behavior patterns.

This project also serves as a showcase of my skills in handling real-world data, preprocessing, model building, and communicating insights clearly — qualities that are essential in a data science or quant role.
